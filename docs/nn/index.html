Du treng ein ElevenLabs-nøkkel (krevst) og eventuelt ein Gemini-nøkkel for nynorsk-omsetjing. Begge har gratis nivå.

Bygd med

Lett, nativ, privat

Ein rein Swift-app — ikkje Electron, ikkje nettlesar, ingen telemetri. Lyd vert sendt til ElevenLabs for transkripsjon og eventuelt til Gemini for omsetjing. Ingenting anna.

🍎 Swift & SwiftUI 🎙 ElevenLabs Scribe v2 ✦ Gemini 2.5 Flash 📓 Obsidian URI 🔑 macOS Nøkkelring ♿️ Accessibility API

Krav

macOS 14 (Sonoma) eller nyare · Mikrofon · Internett-tilgang
